# Claude Gateway v0.1.0

> 2026-07-23

---

## 新增功能
**自动更新：** 设置 → 更新 → 检查更新。从 GitHub Release 自动下载最新版，一键安装。默认开启启动时自动检查。
**系统状态：** 设置 → 系统 → 每 3 秒自动刷新运行时长、进程数、余额。点击 ↻ 手动刷新余额。

## 修复
- **WinError 2：** `run.bat` 启动时路径补全 npm 目录，避免找不到 claude
- **查看日志无反应：** 折叠展开时高度锁死，改为自适应
- **版本号显示：** 首次展开设置面板时不再空白
- **通知动画：** 修复淡入不生效，新增左侧滑入效果

## 改进
- 放行权限标签改为「放行权限」，副标题精简为「批准所有操作」
- 工具状态栏映射新增 13 种工具（WebSearching、Executing skill、Planning 等）
- 系统信息不再重复显示端口/余额/运行时长

## 安全
- 登录爆破防护（5 次/60s 锁定 5s）
- 文件下载 HMAC 签名短期 token
- 上传文件 magic bytes 校验
- 安全响应头（CSP、X-Frame-Options 等）
